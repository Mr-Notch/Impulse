## Impulse Core
一个专为红石服设计的原版扩展服务端核心

## 支持的内容
- Fabric-API
- Fabric-Carpet
- QuickFunctions 2（Deluxe）
- MCD或MCDR

## 构建方法
暂无

## 和原版服或MCDR比较
- 原版端：不支持MCDR插件
- Impluse端：支持90%的MCDR插件
- MCDR守护端：安全性有待提升，不支持加密守护，强行截获数据包
- Impulse端：MCDR插件API监听全在内部进行，无需额外外置其他程序
